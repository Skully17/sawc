Markdown

Markdown is a basic markup language that is mostly used for simple documents that do not need to look fancy. This would be things like "README" files, simple webpages, notes, etc.

Brief Background

Markdown was created in 2004 by John Gruber. When it was released it was very basic so many people had to modify it so it suited their purpose. Markdown is now one of the most popular markup languages, probably due to how simple and easy it is to use.
